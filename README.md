# Cake Order Website

### Goal: 
- Practise python and flask

## Installation
- cd into file
- ```pipenv shell``` to run pipenv
- ```pip install -r requirements.txt``` to install dependencies

## Usage
- ```pipenv shell``` to run pipenv 
- ```python main.py``` to run development
